# Forsah
Forsah is a web application that connect between freelancers and clients with all features that needs to make a deal and make some business with each other.
support credit card and paypal payment methods with ability to make chatting and negotiating to achive desired goals 

## Languages
- Node.js
- React.js
- Tailwindcss
- SQL
- other desgin frameworks
